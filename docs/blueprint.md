# **App Name**: Bhanuteja's Portfolio

## Core Features:

- Hero Section: Displays Nallamothu Bhanuteja's name, role, tagline, intro, CTA buttons, and social icons with an animated background.
- About Section: Presents a professional summary, focus areas, and skill highlights in an engaging manner.
- Skills Section: Showcases categorized skills in frontend, backend, databases, cloud & DevOps, and languages using modern icons and hover effects.
- Projects Section: Displays project showcase cards with screenshots, titles, descriptions, tech stack tags, and links to live demos and GitHub repositories.
- Experience Section: Presents a timeline or vertical cards detailing professional experience, impact metrics, and responsibilities.
- UI/UX Focus Section: Showcases responsive design, performance optimization, smooth animations, modern layouts, and interaction design highlights.
- Contact Section: Includes email, GitHub, LinkedIn links, a contact form with validation, and a resume download option.

## Style Guidelines:

- Primary color: Deep slate (#212121) for a premium developer aesthetic.
- Background color: Dark gray (#333333) with low saturation for contrast.
- Accent color: Gradient from cyan (#48dbfb) to electric violet (#8e44ad) for highlights and CTAs.
- Body and headline font: 'Inter' sans-serif for a modern, clean feel.
- Note: currently only Google Fonts are supported.
- Use modern, minimalist icons to represent skills, technologies, and links.
- Spacious, grid-based responsive layout with rounded cards and soft shadows for a clean and premium feel.
- Smooth scroll, section reveal animations, hover elevation, button glow, animated links, and subtle cursor effects.
